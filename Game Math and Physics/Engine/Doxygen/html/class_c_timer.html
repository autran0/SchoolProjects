Detailed Description

The timer.

The timer allows you to manage game events by duration, rather than on a frame-by-frame basis. This simple version of the timer is based on the Windows API function timeGetTime, which is notoriously inaccurate but perfectly adequate for a simple game demo. The timer can be set to step mode, in which case time is paused, and advanced by a fixed amount when asked.

Created by Ian Parberry to accompany his book "Introduction to Game Physics with Box2D", published by CRC Press in 2013. Has interval ms elapsed since start?

The elapsed function is a useful function for measuring repeating time intervals. Given the start and duration times, this function returns TRUE if the interval is over, and has the side-effect of resetting the start time when that happens, thus setting things up for the next interval.
