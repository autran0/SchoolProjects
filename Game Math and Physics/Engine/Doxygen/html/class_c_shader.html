Detailed Description

The shader class.

The shader class takes care of the management and operation of the vertex and pixel shaders.

Created by Ian Parberry to accompany his book "Introduction to Game Physics with Box2D", published by CRC Press in 2013. void CShader::SetShaders ( ID3D11DeviceContext2 *  pDC2)

Set shaders.

Set the renderer's vertex and pixel shaders. This is intended to be called many times a frame, if necessary, and assumes that everything has been correctly initialized. For speed it does no error checking except to a null pointed check on the device context pointer provided as a parameter.

Parameters
pDC2Pointer to a D3D11 device context 2.
