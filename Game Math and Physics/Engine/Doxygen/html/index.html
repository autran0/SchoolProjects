Engine

This basic 2D sprite-based graphics game engine uses Microsoft DirectX 11.2 for graphics, some Windows API functions, and TinyXML2 for reading XML files. It shouldn't be necessary for you to drill down into the code, but you should probably know about the interfaces. This is not about writing game engine code after all, it's about incorporating game physics into a game engine.

Created by Ian Parberry to accompany his book "Introduction to Game Physics with Box2D", published by CRC Press in 2013.
