void CRenderer::EndFrame ( )

End an animation frame.

End of frame processing, meaning that we preent the rendered image in video memory to the monitor screen. This is currently chained to vertical retrace to limit the animation to 60fps max. If you unchain it, bad things happen to the physics engine in the relaxation demo. (Fixing that is left as an exercise for the reader.)
