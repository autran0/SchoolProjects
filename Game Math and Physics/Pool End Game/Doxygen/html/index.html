Pool End Game

This is the Pool End Game, a simple Windows application that uses hand-written physics code based on Euler integration for motion and ball-to-wall and ball-to-ball collision detection and response from Section 2.2. The code is covered in Chapter 3. The aim is to sink the 8-ball while not sinking the cue-ball.

Use the up and down arrow keys to adjust the initial position of the cue-ball, the left and right arrow keys to adjust the cue-ball's direction (indicated by an arrow on the screen). The Page Up and Page Down keys adjust the cue-ball's direction by larger increments than the left and right arrow keys. The space key fires the cue ball unless it's in a pocket, in which case it starts a new game. The F1 key freezes time. Hitting the F2 key then advances time by one frame, so that you can see collision response in detail. Hitting F1 again returns to real time. ESC exits the game. These are summarized below.

Key
Action
ESC Exit
F1 Freeze/unfreeze time
F2 Advance frozen time by one frame
Up arrow Move cue ball upwards on the base line by a small amount
Down arrow Move cue ball downwards on the base line by a small amount
Left arrow Rotate cue ball direction counterclockwise by a small amount
Right arrow Rotate cue ball direction clockwise by a small amount
Page up Rotate cue ball direction counterclockwise by a larger amount
Page down Rotate cue ball direction clockwise by a larger amount
Space Fire cue ball or start a new game if it is in a pocket
